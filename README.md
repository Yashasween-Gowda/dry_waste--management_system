# Dry Waste Management System

## 📌 Description
This project is a web-based application designed to manage dry waste efficiently. It allows users to register, place orders for waste collection, and track their waste processing.

## 🚀 Features
- User Registration & Login
- Waste Collection Requests
- Order Processing System
- Admin Management (if any)
- Product/Waste Handling

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## 📂 Project Structure
- index.php – Home page
- login.php – User login
- register.php – User registration
- process_waste.php – Waste processing
- product_images/ – Images folder
- uploads/ – Uploaded files
